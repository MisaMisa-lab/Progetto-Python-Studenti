# Progetto Python Studenti

Questo progetto nasce come applicazione Python pensata per aiutare gli studenti universitari a organizzare meglio studio, esami e lezioni.

L’idea è avere un’unica applicazione che raccolga strumenti diversi ma collegati tra loro, invece di usare mille app separate.



L’applicazione va avviata da tabsapp.py.

Questo file è il punto di partenza dell’intero progetto: inizializza l’interfaccia grafica e carica tutte le varie sezioni (tabs).



Esistono anche altri file che possono essere eseguiti singolarmente, ad esempio pomodoro.py, ma servono principalmente per usare una versione limitata da riga di comando.

Per vedere l’applicazione completa l’avvio corretto è sempre tabsapp.py.



Struttura generale

L’applicazione è divisa in più tab, ognuna con una funzione diversa.

Tutti i dati inseriti possono essere ordinati.



Tab Esami

Qui è possibile inserire gli esami universitari con voto, crediti e data.

L’app calcola automaticamente la media aritmetica e la media ponderata.

Gli esami vengono salvati e possono essere ordinati per data, voto o nome.



Tab Pomodoro

Questa sezione è dedicata allo studio tramite il metodo Pomodoro.

Permette di avviare sessioni di studio e salva automaticamente lo storico.

È possibile vedere quanto si è studiato per giorno, per settimana e per mese.



Tab Schedule

È una semplice agenda per organizzare le lezioni universitarie.

Mostra quali lezioni ci sono, in che giorno e a che ora.



Tema

L’applicazione supporta sia il tema chiaro che quello scuro.

Il tema viene applicato a tutta l’interfaccia grafica.



Salvataggio dei dati

Tutti i dati inseriti, come esami, sessioni di studio e lezioni, vengono salvati in un database.



Perché esiste questo progetto

Questo progetto è nato sia come strumento utile per la vita universitaria, sia come modo per mettere in pratica concetti importanti di Python.

In particolare l’organizzazione del codice in moduli, la separazione delle responsabilità, la gestione dei dati persistenti e lo sviluppo di una interfaccia grafica strutturata.
